# FAD
Training and evaluation of "Flow-Accelerated Diffusion Model for Trajectory Generation and Optimization in Offline RL". 

## Prerequisites
1. __Download D4RL__
Follow directions from [this link](https://github.com/Farama-Foundation/D4RL)
2. __Download MuJoCo__
Follow directions from [this link](https://github.com/openai/mujoco-py)


## Installation
```
pip install -r requirements.txt
```

## Train
```
python main.py
```

## Evaluation
```
python eval_main.py
```
